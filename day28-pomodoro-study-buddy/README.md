# Pomodoro Study Buddy

A minimalist Pomodoro timer built with Python and Tkinter, designed to help you study

## Overview

Pomodoro Study Buddy follows the classic Pomodoro Technique:

- 25 minutes of focused work
- 5 minute short breaks
- 20 minute long break after every 4 sessions

---

## Features

- Start / Reset functionality
- Automatic work/break cycle
- Long break after 4 sessions
- Live countdown timer
- Visual progress tracking (✓ checkmarks)
- Prevents multiple overlapping timers

---

## Tech Stack

- Python 3
- Tkinter (GUI)
- Standard library (math, time handling)

---

## Download (Windows)

You can download and run the app without installing Python:

👉 **Download:** `PomodoroStudyBuddy.zip`

### How to run:
1. Download the `.zip`
2. Extract the folder
3. Open the folder
4. Run: PomodoroTimer.exe

Note: Windows may show a warning for unsigned applications. This is normal for small projects and open-source tools.
