# Rock Paper Scissors Game

A simple Python implementation of the classic Rock Paper Scissors game.

## Description

This project allows a user to play Rock Paper Scissors against the computer.
It uses Python's random module to generate the computer's moves and compares them to the user's input to determine the winner.

## How to Play

1. Run the Python script:

       python rock_paper_scissors.py

2. Enter your choice when prompted: rock, paper, or scissors.

3. The computer will randomly select its choice, and the winner will be displayed.

## Features

- Play against the computer
- Input validation to prevent incorrect entries
- ASCII art
