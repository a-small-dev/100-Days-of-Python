# Pong Game (Python Turtle)

A classic Pong Game built using Python's built-in turtle graphics module.  
The project is structured using OOP with separate modules for the ball, paddles, scoring, and game logic.

---

## Features

- Two-player gameplay  
- Ball bouncing off walls and paddles  
- Score tracking  
- First player to reach 3 points wins  
- Smooth animation using screen.tracer(0)  
- object-oriented project structure  

---

## Project Structure

- main.py  
- gameball.py  
- paddleone.py  
- paddletwo.py  
- score.py  
- input_handler.py  

---

## Controls

| Player       | Keys            |
|--------------|----------------|
| **Player 1** | Up Arrow → Move Up |
|              | Down Arrow → Move Down |
| **Player 2** | W → Move Up |
|              | S → Move Down |
