# OOP Coffee Machine

An object-oriented implementation of a coffee machine program.  
Users can order drinks, view reports, and the machine automatically manages resources and payments.

This project is based on the classic Coffee Machine exercise, rewritten with classes to model each component.

---

## Features

- Fully object-oriented structure:
  - `Menu`
  - `CoffeeMaker`
  - `MoneyMachine`
- Choose drinks interactively
- View machine resource reports
- Automatic resource checking
- Automatic payment validation
- Clean separation of logic across multiple modules

---

## Installation

Clone the repository:

```bash
git clone https://github.com/a-small-dev/100-Days-of-Python.git
```

Enter the project folder:

```bash
cd day16-oop-coffee-machine
```

Ensure the following files are present:

- `main.py`
- `menu.py`
- `coffee_maker.py`
- `money_machine.py`

---

## How to Run

```bash
python main.py
```
